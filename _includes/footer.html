Made for fun by broccolini

The Robotodex started with some robot illustrations for a friend's talk, which inspired the idea of illustrating an entire army of robots. Since July 2013, an illustrated robot has been posted weekly (or at least regularly), in an effort to build an army of at least 100 hundred robots.

If you have an idea for a robot you'd like to see in this army, send me a tweet or add an issue on github.
